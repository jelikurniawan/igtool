# How to install
In linux/git-bash windows:<br/>

* git clone https://github.com/irfansetyawan/instajram-robot.git
* cd instajram-robot
* npm install
* npm update
* npm audit fix ( on Termux & c9.io )
* Command -> node v1.js -> to start FFT
* Command -> node v2.js -> to start UNFOLLOW NOT FOLLOWBACK
